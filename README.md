# Email Signature Generator for Optimite

## Overview
The Email Signature Generator for Optimite is a tool designed to create professional email signature. 

## Features
- **Social Media Integration**: Add links to your social media profiles.
- **HTML Output**: Generate HTML code for easy integration into email clients.

## How to Use
1. **Input Information**: Enter your name, title, company name, phone number, and any other relevant details.
2. **Preview Signature**: View a live preview of your signature as you make changes.
3. **Copy and Paste**: Copy the generated code and paste it into your email client's signature settings.
